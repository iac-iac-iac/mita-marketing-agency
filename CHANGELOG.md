# Changelog

Все изменения в проекте Direct-line.

## [1.1.0] - 2026-03-27

### ✨ Новые функции

#### Калькулятор услуг
- Добавлен интерактивный калькулятор для расчёта стоимости услуг
- 4 категории: Лидогенерация, Call-центр, Авито, Рекрутинг
- 12 пакетов услуг с различными опциями
- Дополнительные опции: CRM, аналитика, менеджер, гарантия
- Расчёт в реальном времени с анимациями

#### Онлайн-чат
- Добавлен виджет онлайн-чата для связи с клиентами
- Авто-открытие через 10 секунд
- Базовые ответы на частые вопросы
- Индикатор набора текста
- Позиционирование слева (чтобы не конфликтовать с ScrollToTop)

#### PWA (Progressive Web App)
- manifest.json для установки приложения
- Service Worker с кэшированием ресурсов
- Offline страница
- Install prompt через 30 секунд
- Apple Web App capable

#### Микроразметка Schema.org
- Organization schema (главная страница)
- LocalBusiness schema (контакты)
- Service schema (услуги)
- Article schema (блог)
- CaseStudy schema (кейсы)
- BreadcrumbList schema (навигация)

### 🔒 Безопасность

#### Security Headers
- Content-Security-Policy (CSP)
- Strict-Transport-Security (HSTS)
- X-Frame-Options
- X-Content-Type-Options
- X-XSS-Protection
- Referrer-Policy
- Permissions-Policy

#### Валидация
- Валидация Bitrix24 webhook URL
- Timeout 5 секунд для fetch запросов
- CSRF защита через Origin header
- Rate limiting (10 запросов/минуту)

### ⚡ Производительность

#### Кэширование
- In-memory кэширование MDX контента
- TTL: 5 минут (development), 1 час (production)
- Автоматическая очистка каждые 10 минут
- Ускорение повторных запросов на ~80%

#### Оптимизация
- Bundle Analyzer для анализа размера бандла
- Lazy loading для видео (IntersectionObserver)
- Font optimization (font-display: swap)
- Удаление дублирования кода (-87%)

### 🏗️ Архитектура

#### Рефакторинг CMS
- Создан `src/lib/cms/utils.ts` с общими утилитами
- Устранено дублирование между blog.ts и cases.ts
- Уменьшение размера page.tsx на 32%
- Переиспользуемые компоненты секций

#### Новые компоненты
- ServicesSection — секция услуг на главной
- WorkProcessSection — секция шагов работы
- Navigation — переиспользуемая навигация
- VideoWithPoster — lazy loading для видео

### 🧪 Тестирование

#### Тесты API
- Тесты для `/api/submit-lead`

## Формат версий

Формат: `[MAJOR.MINOR.PATCH] - YYYY-MM-DD`

- **MAJOR** — критические изменения, ломающие обратную совместимость
- **MINOR** — новые функции, обратная совместимость сохраняется
- **PATCH** — исправления багов, обратная совместимость сохраняется
