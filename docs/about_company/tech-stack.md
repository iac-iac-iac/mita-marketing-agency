# Technology Stack

## Overview

М.И.Т.А. использует современный стек технологий для обеспечения лидогенерации, автоматизации бизнес-процессов и разработки внутренних инструментов.

---

## Development

### Языки программирования

| Язык | Версия | Использование |
|------|--------|---------------|
| **Python** | 3.x | Основная разработка, скрипты автоматизации, парсинг |
| **JavaScript** | ES6+ | Фронтенд, скрипты, автоматизация |
| **TypeScript** | (опционально) | Типизированный код для сложных проектов |

---

### Фреймворки и библиотеки

| Технология | Назначение |
|------------|------------|
| **React** | Создание пользовательских интерфейсов |
| **Node.js** | Серверная разработка (при необходимости) |

| СУБД | Использование |

## Integration Architecture

### Упрощённая схема интеграций

```
[Внешние источники]
         │
         ↓
[Парсинг / Сбор данных]
         │
         ↓
[IT-аналитик] → [Автоматизация / Обработка]
         │
         ↓
[Битрикс24] ← [Загрузка лидов]
         │
         ↓
[Отдел продаж / КЦ] → [Обзвон → Квалификация → Продажа]
         │
         ↓
[Отчётность / Аналитика]
```
