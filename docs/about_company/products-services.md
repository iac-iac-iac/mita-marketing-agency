## Product: Лидогенерация (Флагманский продукт)

### Description
IT-сервис для поиска номеров телефонов людей, интересующихся продуктами клиента, с последующей обработкой колл-центром для отбора "живых" лидов.

### Target Segment
- Малый бизнес
- Средний бизнес
- Крупный бизнес

**Критерий:** Средний чек клиента от 30 000 рублей

**Приоритетные отрасли:**
- Автосалоны
- Строительство
- Другие отрасли (кроме: Здоровье, Красота)

### Key Features

1. **Поиск лидов**
   - Определение номеров с сайтов конкурентов
   - Парсинг открытых источников
   - Анализ целевой аудитории

2. **Первичная обработка**
   - Автоматизированный сбор данных
   - Фильтрация некорректных номеров
   - Проверка актуальности контактов

3. **Обзвон колл-центром**
   - Профессиональные операторы
   - Готовые скрипты продаж
   - Квалификация лидов

4. **Предоставление результата**
   - Готовые "тёплые" лиды
   - Отчётность по результатам
   - Интеграция с CRM клиента

### Pricing

| Продукт | Объём | Цена (руб.) | Конверсия |
|---------|-------|-------------|-----------|
| **Платный тест** | 100 звонков | 10 000 | ~10% |
| **Тариф "Стандарт"** | 1000 звонков | 148 000 | ~10% |

### Process Flow

```
[Я] Загрузка лидов в Битрикс24
         ↓
[Отдел продаж ГЦК] Холодные звонки → Предложение Платного теста (10 000 руб.)
         ↓
[Клиент] Оплата теста
         ↓
[Отдел тестов] Обработка заказа → 100 звонков
         ↓
[Отдел тестов] Передача результата клиенту
         ↓
[Отдел продаж] Предложение основного тарифа (148 000 руб.)
         ↓
[Клиент] Оплата тарифа
         ↓
[КЦ] Постоянная обработка → 1000 звонков
         ↓
[Клиент] Получение готовых лидов
```

### Competitive Advantages

- **IT-сервис:** Автоматизированный поиск лидов с сайтов конкурентов
- **Собственный колл-центр:** Контроль качества на всех этапах
- **Платный тест:** Возможность протестировать услугу перед покупкой тарифа
- **Прозрачная конверсия:** ~10% из звонка в лид
- **Интеграция с Битрикс24:** Удобная передача лидов в CRM

## Product: Call-центр

### Description
Профессиональный обзвон номеров клиента для получения горячих лидов.

### Target Segment
- Клиенты, покупающие услугу лидогенерации
- Клиенты, нуждающиеся в аутсорсе обзвона

### Key Features

1. **Холодные звонки**
   - Обзвон по базе компании
   - Первичная квалификация
   - Назначение встреч/презентаций

2. **Тёплые звонки**
   - Обработка входящих заявок
   - Консультирование клиентов
   - Дожим до покупки

3. **Скрипты продаж**
   - Разработка индивидуальных сценариев
   - Адаптация под продукт клиента
   - Постоянное улучшение на основе статистики

### Pricing
Входит в состав услуги лидогенерации.

### Competitive Advantages

- **Опытные операторы:** Профессиональные скрипты и обучение
- **4 отдела КЦ:** ~70-80 сотрудников для масштабирования
- **Контроль качества:** Запись разговоров, анализ эффективности

## Product: Авито

### Description
Бюджетный канал привлечения клиентов через платформу Авито, основанный на опыте партнёров.

### Target Segment
- Малый бизнес
- Средний бизнес
- Локальные компании

### Key Features

1. **Публикация объявлений**
   - Создание продающих текстов
   - Подбор фотографий
   - Оптимизация под поиск Авито

2. **Ведение аккаунта**
   - Регулярное обновление объявлений
   - Ответы на сообщения
   - Модерация отзывов

3. **Аналитика**
   - Отслеживание просмотров
   - Анализ конверсии
   - Оптимизация кампаний

### Pricing

| Регионы продвижения | Цена (руб./мес) |
|---------------------|-----------------|
| Менее 4 регионов (не МСК/СПБ) | 30 000 |
| МСК и/или СПБ и/или более 5 регионов | 40 000 |

### Competitive Advantages

- **Бюджетный канал:** Низкая стоимость лида по сравнению с контекстной рекламой
- **Опыт партнёров:** Использование проверенных стратегий
- **Масштабируемость:** Работа в любом количестве регионов

## Product: Рекрутинг

### Description
Процесс поиска, привлечения, отбора и найма сотрудников для вакантных позиций в компании. Предоставляется как внутренняя услуга, так и для внешних клиентов.

### Target Segment
- Компании, нуждающиеся в массовом подборе персонала
- Бизнесы с высокой текучестью кадров
- Клиенты, покупающие другие услуги Direct-line

### Key Features

1. **Поиск кандидатов**
   - Размещение вакансий на площадках
   - Активный поиск (sourcing)
   - Привлечение через колл-центр

2. **Первичный отбор**
   - Телефонное интервью
   - Проверка соответствия требованиям
   - Квалификация кандидатов

3. **Составление сценария**
   - Разработка скриптов для обзвона
   - Адаптация под вакансию клиента
   - Обучение операторов

4. **Работа call-центра**
   - Массовый обзвон кандидатов
   - Назначение собеседований
   - Сопровождение до выхода на работу

### Pricing

| Тариф | Количество кандидатов | Цена (руб.) | Стоимость за кандидата |
|-------|---------------------|-------------|------------------------|
| **Тест** | 10 кандидатов | 10 000 | 1 000 руб. |
| **Пилот** | 100 кандидатов | 130 000 | 1 300 руб. |
| **Стандарт** | 300 кандидатов | 355 000 | 1 183 руб. |
| **Премиум** | 1000 кандидатов | 1 080 000 | 1 080 руб. |

**Включает:**
- Работа call-центра
- Составление сценария обзвона
- Первичный отбор кандидатов

### Process Flow

```
[Клиент] Заявка на подбор → Описание вакансии
         ↓
[Direct-line] Составление сценария
         ↓
[Call-центр] Обзвон потенциальных кандидатов
         ↓
[Отдел рекрутинга] Первичный отбор (10/100/300/1000 кандидатов)
         ↓
[Клиент] Получение базы отобранных кандидатов
```

### Competitive Advantages

- **Собственный колл-центр:** Массовый и быстрый обзвон
- **Гибкие тарифы:** От теста (10 кандидатов) до премиума (1000 кандидатов)
- **Оптовая цена:** Снижение стоимости за кандидата при больших объёмах
- **Комплексный подход:** От сценария до финального отбора

## Service Level Agreement (SLA)

### Лидогенерация
- **Загрузка лидов:** Ежедневно, бесперебойно
- **Качество данных:** Проверка на корректность
- **Отчётность:** По запросу клиента

### Call-центр
- **Время запуска:** 1-3 дня после получения базы
- **Часы работы:** 9:00 — 18:00 (по местному времени)
- **Контроль качества:** Запись разговоров

### Авито
- **Публикация объявлений:** В течение 1-2 дней
- **Обновление:** Ежедневно/еженедельно (по тарифу)
- **Отчётность:** Ежемесячно

### Рекрутинг
- **Срок подбора:** Зависит от тарифа и вакансии
- **Качество кандидатов:** Соответствие требованиям клиента
- **Замена:** В случае несоответствия (условия по договорённости)
