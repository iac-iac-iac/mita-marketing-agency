## 📋 О проекте

**Статус:** ✅ Завершён (24 марта 2026)  
**Технологии:** Next.js 14, TypeScript, Tailwind CSS, Framer Motion

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск (development)
npm run dev

# Сборка (production)
npm run build
npm start
```

**Сайт:** http://localhost:3000
