# Безопасность данных в Direct-line

Мы понимаем, насколько важна защита данных для вашего бизнеса. Direct-line использует современные технологии и процессы для обеспечения максимальной безопасности информации.

---

## Хранение и обработка данных

### Сбор информации

Direct-line использует автоматизированные системы для сбора и обработки лидов:

- **Парсинг открытых источников** — сбор данных с сайтов конкурентов и публичных ресурсов
- **Автоматическая фильтрация** — проверка корректности номеров и email
- **Валидация данных** — исключение некорректных и устаревших контактов

### Хранение данных

Все данные хранятся в защищённых базах данных с соблюдением следующих принципов:

- **Локальное хранение** — основные данные размещаются на локальных серверах компании
- **Регулярное резервное копирование** — еженедельное и ежемесячное备份
- **Разграничение доступа** — доступ к данным предоставляется по ролям сотрудников

### Обработка данных

Процесс обработки лидов включает несколько этапов:

1. **Загрузка в Битрикс24** — автоматическая загрузка лидов в CRM-систему
2. **Первичная обработка** — проверка на корректность и актуальность
3. **Передача в отдел продаж** — распределение по менеджерам
4. **Обзвон колл-центром** — квалификация и отбор "живых" лидов

## Юридические гарантии

### Конфиденциальность

Direct-line гарантирует конфиденциальность данных клиентов:

- **Неразглашение** — не передаём данные третьим лицам
- **Внутреннее использование** — данные используются только для оказания услуг
- **Соглашения о конфиденциальности** — все сотрудники подписывают NDA

### Соответствие требованиям

Мы соблюдаем требования законодательства:

- **Федеральный закон № 152-ФЗ** — о персональных данных
- **Политика конфиденциальности** — опубликована на сайте
- **Пользовательское соглашение** — регламентирует оказание услуг

### Безопасность данных

Принимаемые меры защиты:

- **Физическая защита серверов** — контролируемый доступ к оборудованию
- **Защита от несанкционированного доступа** — пароли, двухфакторная аутентификация
- **Регулярные аудиты** — проверка систем безопасности

## FAQ по безопасности

### Как вы обеспечиваете безопасность моих данных?

Мы используем многоуровневую систему защиты: локальное хранение данных, разграничение доступа, регулярное резервное копирование и мониторинг безопасности.

### Где хранятся данные?

Основные данные хранятся на локальных серверах компании в г. Саратов. Доступ к серверам имеют только уполномоченные сотрудники IT-отдела.

### Используете ли вы облачные решения?

В настоящее время мы используем локальные сервера. Облачные решения планируются к внедрению для масштабирования и обеспечения высокой доступности.

### Как долго хранятся данные?

Срок хранения данных определяется договором с клиентом. После окончания сотрудничества данные могут быть удалены по запросу клиента.

### Кто имеет доступ к данным?

Доступ предоставляется по ролям:
- Операторы колл-центра — только текущие лиды для обзвона
- Менеджеры — данные своих клиентов
- Руководители — расширенный доступ в рамках отдела
- IT-отдел — технический доступ для поддержки систем

### Что происходит с данными после окончания сотрудничества?

По умолчанию данные архивируются. Клиент может запросить полное удаление своих данных в любое время.

### Используете ли вы данные для собственных нужд?

Нет, данные клиентов используются исключительно для оказания услуг в рамках договора.
