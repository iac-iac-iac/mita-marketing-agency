### Оптимизация изображений

**Рекомендации:**
1. Конвертировать Favicon в `.ico` формат (16x16, 32x32, 48x48)
2. Сжать все PNG через TinyPNG или ImageOptim
3. Для веба использовать формат WebP (с fallback на PNG)
4. Добавить lazy loading для изображений ниже первого экрана

### Next.js Image компонент

```tsx
import Image from 'next/image';

// Hero-баннер
<Image
  src="/images/hero-banner/Hero-banner_main_link.png"
  alt="Direct-line funnel visualization"
  fill
  className="object-cover"
  priority
/>

// Иконка
<Image
  src="/images/icons/service_icon/1_service_icon_Lead_generation.png"
  alt="Lead generation"
  width={64}
  height={64}
/>
```
