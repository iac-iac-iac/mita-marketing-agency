### Что можно упростить

1. **Онлайн-чат**: Использовать готовое решение (Tawk.to, Crisp) вместо кастомного
   - Экономия: ~16 часов

2. **PWA**: Использовать next-pwa без кастомизации
   - Экономия: ~8 часов

3. **Image CDN**: Отложить или использовать Vercel Image Optimization
   - Экономия: ~6 часов

4. **Email уведомления**: Отложить до получения SMTP доступа
   - Экономия: ~3 часа

5. **Развёрнутая API документация**: Использовать простой README вместо Swagger
   - Экономия: ~6 часов
